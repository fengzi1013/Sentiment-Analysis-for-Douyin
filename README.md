# Sentiment-Analysis-for-Douyin
> 1,使用spider中代码爬取抖音短视频中评论，点赞数,用户id等信息，供后续使用。
>
> 2,使用BERT等预训练模型，针对抖音评论进行简单的多分类情感分析任务。

> + 情感分类数据格式
>
>   | text                                                         | label |
>   | ------------------------------------------------------------ | ----- |
>   | 今天晚上跟着你俩运动了一会，感觉两教练很有正能量，原来你们就是最近很火的踢毽子舞教练[比心] | 1     |
>   | 一点都不喜欢那个台湾腔，不喜欢，没多漂亮                     | -1    |
